# Disabling and Deleting User Accounts with Active Directory

### For the this demonstration, I created a new OU called "Disabled Users" for holding user accounts that have been disabled in the Contoso OU.
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/1.png)
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/2.png)
### Select and right-click a user object. You shall find an option to disable the account.
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/3.png)
### Clicking the option shall confirm the disabling of the account. It is recommended that disabled accounts be moved to a designated OU for organizational purposes. 
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/4.png)
### Click and dragging the user object to the Disabled Users OU will move the user object to that respective OU. Alternatively you may right-click and select the "move" option to move the selected object to a specified OU.
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/5.png)
### Trying to log into a disabled account will show the following screen.
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/6.png)
### To re-enable the disabled account, right-click the selected disabled account and select the "Enable Account" option.
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/7.png)
### Enabling an account will show the following confirmation.
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/8.png)
### To delete an account, right-click the selected user account object and select the "delete" option.
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/9.png)
### Selecting the option will show confirmation window asking if you are certain that you wish to delete the selected user. 
![DD](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Disabling%20and%20Deleting%20User%20Accounts%20with%20Active%20Directory/sub/10.png)
