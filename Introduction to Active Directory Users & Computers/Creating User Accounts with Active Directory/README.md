# Creating User Accounts with Active Directory

### From the Server Manager, go to Tools > Active Directory Users and Computers. Create an new Organizational Unit. You can name it anyway you want.
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/1.png)

### I named mine "Contoso".
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/2.png)

### Within the new OU, create two new OUs. One for users and one for computers.
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/3.png)

### In the users OU, create a new user profile
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/4.png)

### Give it both a name and logon name. Mine uses the first initial and lastname convention for logon.
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/5.png)

### For the purposes of being a lab exercise, a simple password is recommended.
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/6.png)

### Click "Finish."
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/7.png)
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/8.png)

### From the properties of the user profile, you can see that it is a member of Domain Users.
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/9.png)

### To test the newly created user profile. we shall login to the Client1 workstation that is part of the Domain.
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/10.png)

### It works!
![UA](https://github.com/whuynhit/ActiveDirectory/blob/main/Introduction%20to%20Active%20Directory%20Users%20%26%20Computers/Creating%20User%20Accounts%20with%20Active%20Directory/sub/11.png)
